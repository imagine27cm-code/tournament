# ✅ Пошаговая инструкция для Supabase
Делай строго по порядку, один пункт за другим

---

## 🎯 Шаг 1: Создаем проект на Supabase
1.  Открой https://supabase.com
2.  Нажми кнопку `Start your project`
3.  Войди через GitHub / почту
4.  Нажми **New project**
5.  Заполни ровно так:
    | Поле | Значение |
    |---|---|
    | Название проекта | `tourney` |
    | Пароль БД | придумай сложный пароль и **сохрани его** |
    | Регион | `Frankfurt (Germany)` |
    | План | Free |
6.  Нажми `Create new project`
7.  ⏳ Подожди 2 минуты пока проект создастся

---

## 🔌 Шаг 2: Получаем строку подключения
1.  Когда проект загрузился, в левом меню нажми:
    `Settings` ➔ `Database`
2.  Прокрути вниз до блока `Connection string`
3.  Выбери вкладку `URI`
4.  Скопируй эту строку полностью
5.  Открой файл `.env` в твоем проекте
6.  Замени старую строку `DATABASE_URL=` на то что скопировал
7.  Вставь свой пароль вместо `[YOUR-PASSWORD]` в этой строке

---

## ✏️ Шаг 3: Обновляем Prisma схему
1.  Открой файл `prisma/schema.prisma`
2.  Найди блок `datasource db`
3.  Замени его полностью на это:
    ```prisma
    datasource db {
      provider = "postgresql"
      url      = env("DATABASE_URL")
    }
    ```
4.  Сохрани файл

---

## 🚀 Шаг 4: Создаем все таблицы автоматически
Открой терминал в папке проекта и выполни одну команду:
```bash
npx prisma migrate dev --name first-deploy
```

✅ После этой команды:
- Все 10 таблиц автоматически создадутся в Supabase
- Все перечисления, ключи и индексы тоже создадутся
- Структура будет точно повторять твой проект

---

## ✅ Шаг 5: Проверка
Выполни в терминале:
```bash
npx prisma studio
```

Откроется панель, в ней ты увидишь все твои таблицы. Готово!

---

## ❗ Важно:
✅ Твой существующий код не нужно менять вообще
